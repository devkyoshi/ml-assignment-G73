## Dataset

- **`loan-dataset.csv`**: The primary dataset used across the models for training, validation, and testing.

- The dataset used in this study is the Loan Prediction Dataset, publicly available on Kaggle at the following link: https://www.kaggle.com/datasets/ninzaami/loan-predication/data
- This dataset was collected by a financial institution and contains historical records of loan applications along with their final approval outcomes. It is widely used as a benchmark dataset for binary classification problems in the financial domain.
